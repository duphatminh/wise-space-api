## Node.js + Expressjs + MongoDB - Building an API for Wise Space App

## Requirements

* **nodejs >= v14.7.0**
* **npm >= v6.14.7**
* **yarn >= v1.19.1**

## Clone project and run 

```
$ git clone https://git_url_clone <project_dir>
$ cd <project_dir>
$ node folder_name/file_name.js
$ (recommend) npm run dev 
```
